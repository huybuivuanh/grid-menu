React is basically all HTML, CSS, JS combined. Elements are separated in components and imported for reusability.

There's a lot of files in the project directory right now but you can ignore most of them because they are just
configuration files.

Files in pages folder are the pages that get rendered when you click on nav bar so you just have to add
components (textarea, font list components what you'll be implementing) to that file.

Files in components folder are smaller components that get added to the pages

FontLoading.js: Functions that fetch and load fonts from google font through google font API. Import and
use it to load fonts

App.jsx: root of all components/pages that handle pages navigation
